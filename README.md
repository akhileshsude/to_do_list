📝 React To-Do Web App

A simple and responsive To-Do List web application built using React. This project helps users manage daily tasks efficiently with an intuitive UI and smooth interactions — all without a backend.

🚀 Features
✅ Add new tasks
✏️ Edit existing tasks
🗑️ Delete tasks
✔️ Mark tasks as completed
🔍 Filter tasks (All / Completed / Pending)
💾 Local storage support (persists data in browser)
📱 Responsive design for all screen sizes

🛠️ Tech Stack
Frontend: React (Functional Components + Hooks)
Styling: CSS / Tailwind 
State Management: useState, useEffect
